# Mossaka's Hevy Data Analysis

This project is a simple dashboard for analyzing workout data exported from the Hevy app.

## Setup

1. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Make sure your Hevy CSV export files are in the `data/` directory.

## Usage

### Unified Dashboard (Recommended)

Run the unified dashboard script for a complete experience:
```
python run_dashboard.py
```

The dashboard will be available at http://localhost:8000
